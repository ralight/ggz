Documentation of GGZ game protocols

Several GGZ games work with a binary opcode-based communication protocol, which is very efficient, and at the same time easy to understand when using the ggzcomm XML format for defining the protocol and generating the documentation. For several such protocols, the documentation, which was generated automatically using XML style sheet transformation (XSLT), is linked from this file.
