Detailed Description

This is similar to GGZCoreConf, but provides additionally a mode which applies to the way one uses the methods.

enum GGZCoreConfioOptions {readonly = 1, readwrite = 2, create = 4 }

Options which are passed to the constructor as an or'd list.
