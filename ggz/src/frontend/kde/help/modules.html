 GGZ Help - Game modules
Theoratically, every game, even every other program can be launched from within the GGZ. In practice it wouldn't make sense to connect to a game server, launch a game and tell all login information again.

There is a standard library (libggzgame) which should be used if you plan to integrate or migrate an Open Source game project.
