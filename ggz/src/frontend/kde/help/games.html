GGZ Games

 There are currently 7 games available: TicTacToe, NetSpades, LaPocha, Connect the Dots, Hastings1066, Reversi and Sink the Ship. In the future this number will grow, and so will the number of players too, so be sure to learn how to play all the games before the others do!
