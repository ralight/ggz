 GGZ Help - The ggz:// protocol
To enable web-based gaming, a protocol named ggzproto was invented. It handles the following tasks:
-launching games (e.g. ggz://launch.ggzproto/tictactoe),
-user management (e.g. ggz://central.ggzproto/login),
-finding other players (e.g. ggz://community.ggzproto/clan/pokerface).
You can always access the pages per HTTP, e.g. via http://mindx.sourceforge.net/ggz/ggzproto/central/login.php
