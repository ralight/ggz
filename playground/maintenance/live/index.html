:: Server information ::

This server, live.ggzgamingzone.org, runs a GGZ Gaming Zone server (ggzd) on port 5688. Most likely you want to play on it, head over to GGZ Community for an easy start. The server also runs Grubby (the chat bot) and the GGZ metaserver on port 15689.

:: Java applet ::

For security reasons, unsigned Java applets need to be hosted on the same server they connect to during the execution. Therefore, GGZ-Java is made available here. It needs Java 5 to be installed.
